# Object-Oriented Programming

Coursework exercises and practice projects from my Object-Oriented Programming module.

This folder is organised by type so the stronger examples are easier to find and the smaller practice exercises do not make the module look cluttered.

## Structure

```text
object-oriented-programming/
├── case-studies/
├── exercises/
└── patterns/
```

## Case Studies

- `case-studies/atm.py` - ATM and current-account example using classes, validation, and custom exceptions.
- `case-studies/library_management_system.py` - Library and book management example.
- `case-studies/hospital_patient_record.py` - Encapsulation practice with patient records and bank account privacy.
- `case-studies/online_shopping_system.py` - Larger OOP exercise using abstract base classes and an online shopping model.
