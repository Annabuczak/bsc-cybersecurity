# BSc Cyber Security Coursework

This repository organises practical exercises and study notes from my BSc Cyber Security degree at the University of Essex Online.

It includes programming exercises, networking fundamentals, operating systems security, ethical hacking notes, and object-oriented programming practice.

## Structure

- `fundamentals-of-programming/` - Python basics, logic, functions, and early programming exercises.
- `object-oriented-programming/` - Classes, inheritance, error handling, design patterns, and applied OOP exercises.
- `fundamentals-of-networks/` - Networking concepts, communication flow, and practical network study.
- `operating-systems-security/` - Operating system security notes and module work.
- `ethical-hacking/` - Ethical hacking study notes and practical exercises.
- `introduction-to-cyber-security/` - Core cybersecurity concepts, risks, threats, and defensive foundations.

## Purpose

This repo is a study record rather than a single finished application. I use it to keep my coursework organised, track progress, and show the practical skills I am building across the degree.

## Current Focus

- Python programming fundamentals
- Object-oriented programming
- Linux and operating systems security
- Networking fundamentals
- Ethical hacking foundations
- Clear technical documentation
