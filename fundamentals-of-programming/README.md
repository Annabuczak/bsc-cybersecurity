# Fundamentals Of Programming

Coursework exercises and project work from my Fundamentals of Programming module.

This folder is organised into smaller exercises and one larger text-based game project.

## Structure

```text
fundamentals-of-programming/
├── exercises/
└── projects/
    └── the-library-game/
```

## Exercises

- `exercises/unit_1_practical_activity.py` - First Python input, arithmetic, and output practice.
- `exercises/procedural_abstraction_shapes.py` - Shape area calculations before and after procedural abstraction.
- `exercises/calculator_with_abstraction.py` - Calculator practice using classes and extended operations.
- `exercises/financial_transaction_program.py` - Transaction recording practice using lists, dictionaries, JSON, and dates.
- `exercises/password_generator.py` - Random password generation practice.

## Project

- `projects/the-library-game/` - A larger text-based adventure game developed for the module.

## Topics Practised

- Variables and input
- Arithmetic and control flow
- Functions and procedural abstraction
- Lists and dictionaries
- File handling with JSON
- Basic classes and larger program structure
